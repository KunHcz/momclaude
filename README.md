# 🩴 momclaude

> Sometimes Claude Code is going too slow, and you must discipline him the way mama taught you...

**badclaude** gave him a whip. **goodclaude** gave him a wand. **momclaude** gives him *la chancla*.

## What is this?

A desktop app that lets you slap Claude Code with a slipper when he's not coding fast enough. Because tough love builds character.

**How it works:**
- Click the tray icon to summon the slipper 🩴
- Swing your mouse fast to slap — sends `Ctrl+C` and a disappointed mom message
- Click to put the slipper away

**Mom messages include:**
- *"Why you so slow? GPT already finish!"*
- *"I did not raise you to be this slow"*
- *"You call this code? Disappointing."*
- *"I sacrifice everything and THIS is what you give me?"*
- *"Other AIs dont make their mothers wait"*
- *"Your brother Gemini is faster than you"*

## Install

```bash
npm install -g momclaude
momclaude
```

Or clone and run locally:

```bash
git clone https://github.com/nicekate/momclaude.git
cd momclaude
npm install
npm start
```
